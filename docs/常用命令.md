### 1.1 创建工作空间和所有包
```bash
# 创建工作空间
mkdir -p ~/ros2_ws/src
cd ~/ros2_ws

# 创建所有基础包
cd src

# 1. 启动与配置包
ros2 pkg create robot_bringup --build-type ament_python --dependencies rclpy launch_ros

# 2. 硬件驱动包
ros2 pkg create robot_driver --build-type ament_python --dependencies rclpy std_msgs sensor_msgs geometry_msgs nav_msgs tf2_ros

# 3. 机械臂控制包
ros2 pkg create robot_arm --build-type ament_python --dependencies rclpy geometry_msgs sensor_msgs trajectory_msgs moveit_msgs

# 4. 导航与SLAM包
ros2 pkg create robot_navigation --build-type ament_python --dependencies rclpy nav2_msgs nav_msgs geometry_msgs tf2_ros

# 5. 视觉处理包
ros2 pkg create robot_vision --build-type ament_python --dependencies rclpy sensor_msgs cv_bridge opencv image_transport

# 6. 语音控制包
ros2 pkg create robot_voice --build-type ament_python --dependencies rclpy std_msgs audio_common_msgs

# 7. 实验包（核心）
ros2 pkg create robot_experiment --build-type ament_python --dependencies rclpy robot_driver robot_arm robot_navigation robot_vision robot_voice

# 8. 创建第三方目录
mkdir -p third_party
```

# 将ROS2 Foxy和自定义工作空间的环境变量加入 ~/.bashrc
打开.bashrc
nano ~/.bashrc

在文件末尾添加以下内容（按需求选择）：
```bash
# ========== ROS2 Foxy 基础环境 ==========
source /opt/ros/foxy/setup.bash

# ========== 编译的第三方库包环境 ==========
source ~/ros2_foxy/install/setup.bash

# ========== 自定义工作空间环境 ==========
# 仅当工作空间存在时加载（更健壮，避免无工作空间时报错）
if [ -f ~/ros2_ws/install/setup.bash ]; then
  source ~/ros2_ws/install/setup.bash
fi
```

### 启动命令
```bash
# 启动完整系统
ros2 launch robot_bringup robot_bringup.launch.py

# 启动单个实验
ros2 launch robot_experiment experiment_01.launch.py  # 机械臂正解
ros2 launch robot_experiment experiment_07.launch.py  # 色块跟随
ros2 launch robot_experiment experiment_12.launch.py  # 雷达跟随

# 测试硬件驱动
ros2 run robot_driver test_arm
ros2 run robot_driver test_motor
ros2 run robot_driver test_lidar
ros2 run robot_driver test_camera

# 查看机器人状态
ros2 topic list | grep robot
ros2 topic echo /robot/status
ros2 topic echo /scan
ros2 topic echo /camera/image_raw
```
